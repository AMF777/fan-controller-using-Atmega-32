# fan-controller-using-Atmega-32
A temperature-controlled fan using ATmega16 microcontroller, in which the fan is automatically turned ON or OFF according to the temperature.
## Main Components
- ATmege16 
- 2*16 LCD
- LM35
- DC Motor
- L293D

## Drivers
- ADC
- PWM
- GPIO
- LCD
- LM35
- DC Motor

## Tools
- Eclipse
- Proteus
